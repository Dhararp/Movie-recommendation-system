# 🎬 Movie Recommendation System

An AI-powered Movie Recommendation Web Application built using Flask, Python, and MySQL that suggests similar movies based on user selection and tracks search trends with data visualization.

## 📌 Project Overview

The Movie Recommendation System is a web-based application that recommends similar movies based on user selection. 

It uses machine learning techniques to find related movies and stores user search history in a MySQL database. The system also analyzes search trends and visualizes them using Matplotlib regression graphs.

This project combines:
- Machine Learning
- Data Science
- Web Development
- Database Management
- Data Visualization

- ## 🚀 Features

- Movie recommendation based on user selection
- Similar movie suggestion using ML model
- Search history tracking
- MySQL database integration
- Trend analysis using regression graph
- Data visualization using Matplotlib
- Flask-based web interface
- Clean UI with HTML and CSS

## ⚙️ Installation

Step 1: Clone the repository

git clone https://github.com/your-username/movie-recommendation-system.git

Step 2: Go to project folder

cd movie-recommendation-system

Step 3: Install dependencies

pip install -r requirements.txt

Step 4: Setup MySQL database

Create database:

movie_db

Create table:

movie_search_history

Step 5: Run the project

python app.py
